# Simuladores Interactivos Web

Este proyecto contiene dos simuladores desarrollados en **HTML, CSS y JavaScript**:

1. **Simulador de Transferencia de Calor**  
   Calcula la temperatura final de un objeto usando la Ley de Enfriamiento de Newton.

2. **Calculador de Combinaciones (Sorteo)**  
   Calcula el número total de combinaciones posibles en un sorteo tipo Powerball.

## 🚀 Cómo ejecutar
1. Clonar el repositorio.
2. Abrir `index.html` en un navegador.
3. Usar los formularios interactivos.

## 📦 Publicación
El proyecto puede desplegarse en **GitHub Pages**:
- Subir el repositorio.
- Activar Pages en la rama principal.
- Acceder al enlace público generado.
